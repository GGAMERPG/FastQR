# FastQR — десктопный генератор QR-кодов

FastQR — это простая программа для Windows, которая создаёт QR-коды без интернета и без отправки данных в сеть.

---

## Возможности

- Работает без интернета
- Не требует установки Python
- Тёмная тема интерфейса
- Русский и английский языки
- Выбор размера QR-кода (400, 600, 800, 1000 px)
- Сохранение в PNG или JPEG
- Вставка ссылки из буфера обмена одной кнопкой
- Запоминание последней выбранной папки

---

## Установка и запуск

### Готовая программа (для пользователей)

Скачай файл `FastQR.exe` в разделе **Releases** (справа на этой странице) и запусти его.  
Программа не требует установки — просто открой и пользуйся.

> **Примечание:** иконка уже внутри `.exe`, ничего дополнительно скачивать не нужно.
